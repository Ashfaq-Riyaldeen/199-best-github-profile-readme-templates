# Contributing

Welcome to this collection of GitHub profile README templates, and thanks for considering contributing! Contributions are welcome so we can gather the best GitHub profile READMEs in one place.

Please [open a pull request](https://github.com/Ashfaq-Riyaldeen/199-best-github-profile-readme-templates/pulls) if you would like to contribute a new template or improve an existing one. If you would like to comment on or discuss content but don't have concrete changes yet, please [open an issue](https://github.com/Ashfaq-Riyaldeen/199-best-github-profile-readme-templates/issues).

This is a curated list, so the goal is not to collect every possible README but the ones that are most helpful, provide a unique perspective, or are of special significance.

## How to contribute

1. Fork this repository.

2. Clone your fork to your local machine.

3. Create a new branch.

4. Create a new Markdown file inside the **templates** folder. Name the file after your GitHub username:

   ```
   templates/your-username.md
   ```

5. Copy and paste your GitHub profile README content into the new file.

6. At the end of the file, add the following with appropriate changes:

   - A line break using `------`
   - Credit: `[your-username](https://github.com/your-username)`
   - Last edited on: the date you edited this file, in DD/MM/YYYY format.

7. Add a row for your template at the end of the **Templates** table in [README.md](./README.md):

   ```
   | <next number> | [your-username](./templates/your-username.md) |
   ```

8. Commit your changes with a proper message that contains your username, for example:

   ```
   Added template for username: your-username
   ```

9. Push your commit to GitHub.

10. Submit a pull request. All done!

By contributing, you agree to follow our [Code of Conduct](./CODE_OF_CONDUCT.md).
